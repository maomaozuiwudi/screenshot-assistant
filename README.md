# 📸 截图助手 - Screenshot Assistant

> 工具猫系列 - 智能截图助手，高效截图与批注工具

## ✨ 功能

- **9:16 比例截图** — 锁定小红书/抖音等竖屏比例，一键截取完美尺寸
- **分镜描述管理** — 管理多张截图的分镜描述，便于内容规划
- **AI 截图来源建议** — 输入关键词自动推荐最佳截图来源（电商平台、素材网站等）
- **预设尺寸快速切换** — 支持 540×960、720×1280、1080×1920 三种常用尺寸
- **Ken Burns 动画预览** — 内置缩放动画效果预览
- **临时暂存区** — 未确认的截图先暂存，方便筛选和管理

## 🚀 使用

```bash
python screenshot_assistant.py
```

### 依赖

- Python 3.8+
- Pillow
- tkinter（Python 内置）
